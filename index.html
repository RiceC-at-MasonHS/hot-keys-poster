Knowing how to use keyboard shortcuts can seriously speed up your work on any computer.

mostly everywhere

ctrl + A = Select All

ctrl + C = Copy

ctrl + X = Cut

ctrl + V = Paste

ctrl + shift + V = Paste as plain text

ctrl + Z = Undo

ctrl + Y = Redo

ctrl + S = Save

ctrl + P = Print

ctrl + F = Find among text

ctrl + H = Find and Replace

text selection

The cursor for text is called a 'caret' below, since that is the proper name for the blinking vertical line.

home = Jump caret to start of line

end = Jump caret to end of line

backspace = Remove 'char's left of caret

delete = Remove 'char's right of caret

shift + any
arrow = Select from caret

ctrl + any
arrow = Jump caret to next word

left
click + left
click = Select whole word

left
click + left
click + left
click = Select whole line

ctrl + shift + any
arrow = Jump caret by word, while selecting

alt + shift + up/dn
arrow = Move line of text up or down

Chromebooks

+ V = Open clipboard menu

+ L = Lock your screen

ctrl + Shift + Q + Q = Sign Out

ctrl + shift + = Screenshot or screen recording

+ Shift + Space = Emojis and symbols menu

ctrl + Shift + alt + = Barrel-roll your screen

esc + + = Recovery mode

ctrl + + esc = Chromebook Diagnostics

Hold: + + for 10+ seconds = Hard reset

ctrl + alt + / = Check all keyboard shortcuts

Browsers/ Chrome

Space = move/scroll down the page

Space + shift = move/scroll up the page

shift + alt + B = highlight bookmarks bar

ctrl + R = Refresh page (local, from cache)

ctrl + shift + R = Refresh from server (new request to web server)

ctrl + T = Open a new Tab

ctrl + shift + T = Reopen the last browser tab closed

ctrl + J = Downloads page

ctrl + H = Tab history

ctrl + L = Highlights URL

ctrl + TAB = Move to next/ right browser tab

ctrl + TAB + shift = Move to previous/ left browser tab

ctrl + num
1-8 = Jump to specific browser tab in number-order

ctrl + num
9 = Move to the rightmost browser tab, no matter how many tabs

middle
click on a link = Open in and switch to new tab

Hold:ctrl left
click on a link = Open in new tab, in the background

window navigation

ctrl + N = New

ctrl + O = Open

ctrl + W = Close

ctrl + Q = Quit

alt + tab = Switch to previous window

Hold: alt + tab = Window switcher

alt + f4 = close current program (Windows)

ctrl + alt + delete = Task Manager (Windows)

Code Editors

ctrl + / = Comment out for line with caret, or all highlighted text

ctrl + Shift + / = Un-comment for line with caret, or all highlighted text

ctrl + ] = Add tab for line with caret, or all highlighted text

ctrl + [ = Remove tab for line with caret, or all highlighted text

alt + = Move line down

alt + = Move line up

Hold:atl + left
click = Place multiple carets for parallel editing [in VS Code]

Accessibility

alt = Focus on window menu

ctrl + - = Zoom out (Browsers and more)

ctrl + + = Zoom in (Browsers and more)

ctrl + 0 = Reset zoom (Chrome)

ctrl + + H = Invert colors (Chromebook)

Lalt + LShift + Print Screen = High Contrast (Windows)

Shift + alt + L = Highlight Launcher (Chromebook)

ctrl + Shift + = Rotates screen 90 degrees clockwise (Chromebook)

ctrl + Shift + Y = Dictionary lookup (Chromebook)

ctrl + alt + Z = ChromeVox screen reader (Chromebook)

ctrl + + M = Magnify whole screen (Chromebook)

ctrl + + D = Magnify part of your screen (Chromebook)

ctrl + alt + any
arrow = move around in magnifier mode (Chromebook)

ctrl + Shift + Space = Switch keyboard languages (Chromebook)

ctrl + Space = Switch to previous keyboard language (Chromebook)

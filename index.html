Knowing how to use keyboard shortcuts can seriously speed up your work.

Magnifying glass:

mostly everywhere

ctrl + A = Select All

ctrl + C = Copy

ctrl + X = Cut

ctrl + V = Paste

ctrl + Z = Undo

ctrl + Y = Redo

ctrl + S = Save

ctrl + P = Print

ctrl + F = Find among text

ctrl + H = Find and Replace

window navigation

ctrl + N = New

ctrl + O = Open

ctrl + W = Close

ctrl + Q = Quit

ctrl + shift + T = Reopen last window closed

alt + tab = Switch to last window

Hold: alt + tab = Window switcher

text selection

The cursor for text is called a 'caret' below, since that is the proper name.

home = Jump caret to start of line

end = Jump caret to end of line

backspace = Remove 'char's left of caret

delete = Remove 'char's right of caret

shift + arrow
key = Select from caret

ctrl + arrow
key = Jump caret to next word

ctrl + shift + arrow
key = Jump caret by word, while selecting

alt + shift + up/dn
arrow = Move line of text up or down

left
click + left
click = Select whole word

left
click + left
click + left
click = Select whole line

Browsers/ Google Chrome

ctrl + T = Open a new Tab

ctrl + num
1-8 = Jump to browser tab in number-order

ctrl + num
9 = Move to the rightmost browser tab

ctrl + - = Zoom Out

ctrl + + = Zoom in

middle
click on a link = Open link in new tab

ctrl + H = Tab history

ctrl + L = Highlights URL

ctrl + TAB = Move to next/right browser tab

ctrl + shift + TAB = Move to previous/left browser tab

ctrl + R = Refresh page (basic, from cache)

ctrl + shift + R = Refresh from server (new request for content)

space = move/scroll down the page verified (in chrome) shift + space = move/scroll up the page verified (in chrome)

Chromebooks

+ Shift + Space = Emojis and Symbols Menu

ctrl + Shift + Q+ Q = Sign Out

esc + + = Factory reset

ctrl + + esc = Chromebook Diagnostics

esc + + = Recovery mode

Code Editors

Hold:atl + left
click = place multiple carets for parallel editing [in VS Code]

ctrl + / = comment out for line with caret, or all highlighted text

ctrl + Shift + / = un-comment for line with caret, or all highlighted text

ctrl + ] = add tab for line with caret, or all highlighted text

ctrl + [ = remove tab for line with caret, or all highlighted text

'alt' + 'arrow down' = move line down verified (IDEs only, VS-Code verified) 'alt' + 'arrow up' = move line up verified (IDEs only, VS-Code verified)

Accessibility

ctrl + + H = invert colors

Left Alt + Left Shift + PrintScreen = High Contrast (Windows) verified (win/accessibility) ctrl + shift + refresh = rotates screen 90 degrees clockwise verified (Chromebook) `ctrl` + `shift` + `alt` + `refresh` = barrel-roll your screen verified (cb/accessibility)
